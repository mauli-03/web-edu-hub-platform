This page demonstrates the auto-loading black and orange navbar.

How It Works

The navbar is automatically loaded by including the auto-navbar.js script in the head of your HTML document. This ensures a consistent look and feel across all pages.

  • Black background with white text
  • Orange accent colors
  • Responsive design
  • Automatically included on all pages

Features

The navbar includes all the main navigation elements:

  • Home link
  • Education dropdown (Universities, Courses, Resources)
  • Community dropdown (Blogs, Forums, Chat)
  • Tools dropdown (AI, Help Center)
  • User authentication controls

Implementation Instructions

To use the auto-navbar on any page:

<!-- In your HTML <head> tag -->
<script src="../js/auto-navbar.js"></script>

That's it! The navbar will automatically be loaded and styled consistently.
